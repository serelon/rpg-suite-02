# inbox/

Drag-and-drop landing zone for raw material **awaiting mining** — exports, snippets,
screenshots, copied files, half-formed thoughts. Anything not yet processed into a note.

**Rule:** nothing here is canonical. The inbox is a queue, not a store. When an item is
mined into a `docs/notes/` note (or decided to be irrelevant), remove it from the inbox.
A non-empty inbox means there's unprocessed input.
